DATA NORMALIZATION

# Normalisation means data doesn't exceed 1

1. Not Normalised                   
    Age         Salary
    30          4,00,000
    40          60,000
    50          3,00,000

2. Normalised
    Age         Salary
    0.3         1
    0.4         0.15
    0.5         0.75

# 3 METHODS OF DATA Normalisation
    1. Simple Feature Scaling
    2. Min-Max
    3. Z-Score / Standard Score

1. Simple Feature Scaling
    Mathematically,
        Xnew = Xold / Xmax
    Result range: 0 to 1


2. Min-Max
    Mathematically,
        Xnew = ( Xold - Xmin ) /
                              / ( Xmax - Xmin)
     Result range: 0 to 1
    
3. Z-Score / Standard Score
    Mathematically,
        Xnew = (Xold − μ)/ σ        μ = Average of Feature
                                    σ = Standard Deviation
        range: -3 to +3
